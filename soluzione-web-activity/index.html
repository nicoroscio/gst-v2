Nuove Soluzioni e Appliance della linea ACTIV

Sotto il marchio WEB ACTIVITY si inseriscono le nostre proposte sistemistiche:

Firewall > in particolare opensource IpCop

Mail Server > soluzioni Open Source su Linux

Virtual Server > su Windows o su Linux

Centralini IP > in particolare Asterisk che permette una completa gestione da interfaccia web con tante funzionalità che permettono di sfruttare molte possibilità di risparmio offerte dal VOIP.

Abbiamo creato specificatamente una linea di prodotti. software appliance e servizi tutti consigliati e supportati da Gostec e tutti in grado di collocarsi al top nelle rispettive funzionalità:

Vedi i dettagli nelle pagine dedicate:

:: ACTIV.TEL- IL CENTRALINO SOFTWARE IP – E’ un Centralino telefonico Software open source

:: ACTIV.SERVER – Una gamma di server con software open source economici, affidabili e ricchi di funzionalità

:: ACTIV.VSH – Virtual Server Hosting – Virtual Server Hosting un hosting di server virtuale

:: ACTIV.MAIL – Un mail server e agenda aziendale completo open source

:: ACTIV.FIREWALL – Un firewall potente e flessibile basato su kernel linux

:: ACTIV.HOSTING l’Hosting intelligente – Servizio di hosting del dominio con possibilità di gestione autonoma di servizi e applicazioni.
