Setup della posta elettronica Gostec

Server della posta in arrivo (POP3)

Casella di posta gratuita = pop.gostec.net

Casella di posta professionale = pop.gostec.it

Casella di posta su dominio = pop.nome dominio (es. se il dominio è pippo.it il server pop3 sarà pop.pippo.it) o in alternativa pop.gostec.it

Server della posta in uscita (SMTP)

Il server della posta in uscita (SMTP) va configurato in funzione della tipologia di connessione:

Connessione gratuita Analogica/ISDN su numerazione 07211860000 = tmail.gostec.it

Connessione gratuita Analogica/ISDN su numerazione 7090007310 = tmail.gostec.it

Connessione gratuita Analogica/ISDN su numerazione 7020005074 = smtp.free.gostec.net

Connessione professionale Analogica/ISDN su numerazione 7020005074 = smtp.gostec.com

Connessione ADSL dial-up ip dinamico = smtp.adsl.gostec.net

Connessione ADSL dial-up ip fisso e Always-on = smtp.gostec.it

Connessione non Gostec = utilizzare l’indirizzo del server smtp fornito dal provider della connessione

Se è stato attivato l’invio di posta tramite server autenticato (con autenticazione SMTP)

Permette l’invio di mail con qualsiasi connessione, anche non Gostec, ma richiede un dominio proprietario

SMTP.nome dominio (es. se il dominio è pippo.it il server SMTP sarà smtp.pippo.it)

Attivare poi la voce “il server della posta in uscita richiede autenticazione”

Attivare l’opzione “accedi con” e inserire username e password come da contratto.

Per ulteriori necessità di assistenza contatta il reparto tecnico Gostec

CASELLE DI POSTA PEC

Nome account e password come forniti in fase di compilazione del modulo di attivazione

Server della posta in arrivo (POP3): mail.twtcert.it

Impostare la porta 995, attivando anche l’opzione “il server richiede una connessione crittografata (SSL)”

Server della posta in uscita (SMTP): smtp.twtcert.it

Impostare la porta 495, attivando anche l’opzione “il server richiede una connessione crittografata (SSL)”

Attivare l’invio della posta tramite server autenticato: attivare la voce “il server della posta in uscita richiede autenticazione” utilizzando le stesse impostazioni della posta in entrata.
