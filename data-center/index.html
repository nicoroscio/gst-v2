Data Center

GOSTEC dispone di un proprio DATACENTER autonomo per la fornitura dei servizi che utilizzano internet.

Il Datacenter è attualmente composto  di circa 20 server (il circa è perchè i sistemi sono in continua evoluzione).

I server forniscono attualmente i servizi:

  • WEB server Windows
  • WEB server Linux
  • Mail server Windows
  • Mail Server Linux
  • Antispam e antivirus
  • Firewall
  • Backup
  • Faxserver
  • FTP server
  • DNS
  • TACACS e RADIUS Server
  • Virtual server

Tutti i principali servizi sono duplicati o anche triplicati per garantire la massima affidabilità dei servizi forniti.

I locali sono in ambiente climatizzato, l’alimentazione è garantita da gruppi UPS e da gruppo elettrogeno, tutti i sistemi sono protetti da antivirus e firewall.

La connettività è garantita da più linee ridondanti.
